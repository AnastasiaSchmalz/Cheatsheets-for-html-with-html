HTML Tables

In HTML tables consist out of:

  • table

    to create a two-dimensional table

  • thead

    for defining the headings of table columns

  • tbody

    , which contains all the table data

  • tr

    for table rows

  • th

    to give each column a heading

  • td

    for your actual table data

  • tfoot

    for further content or to summarize data

Note, that 'tr' and 'td' are nested elements. This means, they are inside of 'thead', 'tbody' and 'tfoot' and are there for creating rows and cells. Therefore 'td' must be inside of 'tr'!
